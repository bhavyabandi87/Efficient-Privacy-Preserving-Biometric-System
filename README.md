# Efficient Privacy Preserving Biometric System

## Project Description

This project focuses on secure biometric image storage and authentication using cloud computing while preserving user privacy and improving efficiency.

## Technologies Used

* Java
* JSP
* MySQL
* HTML
* CSS
* Apache Tomcat
* NetBeans IDE

## Features

* Biometric image storage
* User authentication
* Privacy preservation
* Cloud-based data management
* Secure database storage

## Project Structure

* `src` - Java source code files
* `web` - JSP and web pages
* `Database.sql` - MySQL Database script
* `Screenshots` - Output screenshots
* `Documentation` - Project documents and reports

## How to Run

1. Import the project into NetBeans IDE.
2. Configure the MySQL database using `Database.sql`.
3. Deploy the project on Apache Tomcat.
4. Run the application.
